HRS Kanban Docs

  1. Kanban board :
    Kanban Board is a project management board where you can assign tasks to people, manage workflow and identify bottlenecks. Place tasks in a column, assign them to members and automate task flow upon certain events
  2. Board :
    The board contains all details related to project and tasks. Assign members, add task priorities, status and add automation. A board can be deleted only by its admin(creator).
  3. Column :
    A column represents a particular set of workflow like a to-do column, completed work column etc.
  4. Task :
    A task is a task assigned to a particular member in the board with a deadline, status, priority assigned. Can be moved to diiferent columns based on rules automatically
  5. Rule :
    A rule is an automation of task flow, where a task upon change in its staus, completion or priority can be moved to a different column automatically.
  6. Members :
    Members are members in board who can assigned tasks or can edit tasks. Members must be registered in the app before adding them to board
